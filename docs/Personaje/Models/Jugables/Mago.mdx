# Mago

La declaración <Highlight color="green">public class Mago extends PersonajeModel</Highlight> <br/> indica que Mago hereda todos los <Highlight color="green">atributos y métodos</Highlight> de la clase abstracta <Highlight color="green">PersonajeModel</Highlight>. Esto garantiza que todo Mago tenga las propiedades esenciales de un <Highlight color="green">personaje</Highlight> (Vida, Ataque, Defensa, etc.).

<JavaBlock title="package [NombreProyecto].Personaje.Models.Jugables">
{`
import Personaje.Models.PersonajeModel;

public class Mago extends PersonajeModel { 
    
    private static final int MagoVida = 1200; 
    private static final int MagoAtaque = 100;
    private static final int MagoDefensa = 30;
    private static final int MagoVelocidad = 70;

    public Mago(String nombre) {
    // Llama al constructor de la superclase (PersonajeModel)
        super(MagoVida, MagoAtaque, MagoDefensa, MagoVelocidad, nombre);
    }   

    @Override
    public void interactuar() {
        System.out.println("El Mago " + this.getNombre() + " canaliza energía arcana.");
    }
    @Override
    public void atacar(PersonajeModel enemigo) {
        // 1. Imprimimos el mensaje de acción
        System.out.println("El Mago " + this.getNombre() + " lanza una bola de fuego a "
        + enemigo.getNombre() + "!");

        // 2. Calculamos el daño efectivo (Daño = Ataque - Defensa del Enemigo)
        int dañoInfligido = this.ataque - enemigo.getDefensa();
    
        // Aseguramos que el daño sea al menos 1, no negativo
        if (dañoInfligido < 1) {
            dañoInfligido = 1; 
        }

        // 3. Aplicamos el daño (Restamos el daño a la vida del enemigo)
        enemigo.setVida(enemigo.getVida() - dañoInfligido);

        // 4. Imprimimos el resultado de la acción
        System.out.println("¡Ha infligido " + dañoInfligido +
         " puntos de daño!");
        System.out.println("La vida restante de " + enemigo.getNombre() +
         " es: " + enemigo.getVida());
    }

    @Override
    public void escapar(PersonajeModel enemigo) {
        System.out.println("El Mago " + this.getNombre() + " se teletransporta lejos de " + enemigo.getNombre());
    }

}
`}
